# Normal distribution lesson copy

<!-- Keep the ## step ids stable. Edit the Markdown below freely. -->

## touch
### instruction
See the blue μ? That's the center. Drag it left or right.

### hint
Drag μ in the formula.

### insight
The whole curve shifts. μ is where the average is — the peak.

## width
### instruction
Now drag σ to make the curve wider or narrower.

### hint
Drag σ upward for wider.

### insight
Bigger σ = more spread. Tiny σ = everyone clustered at the average.

## regions
### instruction
Set σ=1 — the shaded region is 68% of all data.

### hint
Drag σ to 1.0.

### insight
68% within 1σ, 95% within 2σ, 99.7% within 3σ. Works for everything — heights, scores, manufacturing.
